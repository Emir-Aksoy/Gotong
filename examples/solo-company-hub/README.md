# solo-company-hub — 一人公司 hub 开箱包

面向**一人公司**（独立开发者 / 自由职业者 / 独立顾问 / 一人工作室主理人）的
一键装旗舰模板：公司只有你一个人，却要同时干接客、交付、开票、维护资料库、
每天理优先级的活。这个包把「一个人 + 一个贴身管家 + 三个专业分身 + 你的知识库 +
你的现实工具」串成开箱即用的形态，让一个人跑出一个小团队的产能。

配套叙事一页纸见 [`docs/zh/SOLO-COMPANY-HUB.md`](../../docs/zh/SOLO-COMPANY-HUB.md)
（「一个人顶一个团队」+ 开箱验收标准）。

## 装什么

| 件 | 说明 |
|---|---|
| 3 个专业分身 | 客户沟通助手（起草对外沟通，**从不真发**）+ 经营简报员（诚实模式：没数据不编）+ 研究助手（新单该怎么接、报价参考） |
| `client-outreach-approval` 工作流 | 你想对外发点什么 → AI 拟成一页草稿 → **停在你自己的审批步** → 你签发才出最终可发文案，不批就不发。一人公司老板既是发起人也是把关人——**对外动作自己签发** |
| `daily-ops-brief` 工作流 | 每日经营简报（今日优先级 / 待跟进 / 现金流提醒 / 今日一问），`/me` 一键跑；带每早 8 点定时建议（只带节奏不带人，装完「定时」卡补人启用） |
| `new-client-workup` 工作流 | 来了新单/新客户，贴进需求 → 研究助手查你的知识库出一页「这单怎么接」 |
| 4 个连接器槽位 | 知识库 / 日历 / 任务 / 客户资料，全部可选——不挂照样跑，进诚实模式 |
| 2 条 golden-run 验收 | 装完面板「跑验收」零 LLM 判卷（审批流刻意不进验收：它要验的就是「你真签发一次」，见模板头注释） |

三个分身都跑 DeepSeek（`openai-compatible`，`deepseek-chat`）——性价比甜区，
一个人也养得起一个团队的调用量。

## 怎么装

- **面板**：admin →「工作流 / 模板画廊」→「一人公司 hub」→ 安装（导入时填一次 DeepSeek key）。
- **一条命令开荒**（含补人启用定时）：

```bash
gotong provision examples/solo-company-hub/template/solo-company-hub.template.yaml \
  --url http://127.0.0.1:8745 --token <admin-token> --user <你的成员id>
```

## 装完点什么

1. `/me` →「起草一条对外沟通」→ 填想对外发的东西 → 提交（一人公司=发起人即签发人，
   审批卡自动落到你自己的收件箱，不用填 userId）。
2. 你的 `/me` 收件箱出现「对外沟通签发」卡，卡上**内联着草稿全文**（**卡先出现，动作没发生**）。
3. 你签发 → 工作流续跑出最终可发文案；换一次点拒绝 → `finalize` 步被跳过，最终文案为空 = 什么也没发。
4. 「来份今日经营简报」/「研究一个新单」一键跑通即 golden-run 同款路径；
   「定时」卡补人启用后每早 8 点（默认 UTC+8）自动送经营简报到 IM。

## 诚实边界

- `client-outreach-approval` 批准后的产出是**一页可发文案 + 一条记录**，不是真的
  把邮件发出去——模板不携带任何出站凭证，也绝不预授权（**接入 ≠ 授权行动**）。
- 要让 AI 真的替你发邮件 / 发消息 / 开发票，给对应 agent 挂上连接器后，那些动作
  **仍逐次过审批闸**。
- 研究助手的「报价参考」基于你知识库里的历史，**不是市场实时行情**。
- 没挂连接器时进诚实模式：没有的数据就不编（不会伪造「今天的会议」或「某客户欠款 X 元」）。

## 往上兼容（一人公司 → 小团队）

一个人做大了要招人、要接企业级数据源，这套形态是往上兼容的：同一个 hub 加成员、
分角色（`owner`/`admin`/`member`/`viewer`）、把知识库槽位换成企业级后端
（`elasticsearch-kb` 走 [RAG-VIA-MCP](../../docs/zh/RAG-VIA-MCP.md)）。多人 + 多 agent +
企业级数据库/知识库的样板见画廊里的 `cafe-ops`（多角色协作）与
[`docs/zh/SOLO-COMPANY-HUB.md`](../../docs/zh/SOLO-COMPANY-HUB.md) 的「向上兼容」节。
