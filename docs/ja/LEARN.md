# AipeHub を学ぶ（Learn）

<!-- doc-version: 1.0 -->
> **ドキュメントバージョン 1.0** · 日本語訳 · 最終更新 2026-06-27 · 権威ある原典：[English](../../LEARN.md)。訳文と英語版に矛盾がある場合は、英語版が優先されます。

AipeHub を学ぶための最高のコミュニティ制作教材のキュレーションされた陳列館 ——
ビデオ、トーク、チュートリアル、ライトアップ。**ドキュメントを読むより見て学ぶ
方が得意な場合はここから始めてください。**

これは[旗艦テンプレートインデックス](FLAGSHIP-TEMPLATES.md)の**リーチ版**です：
旗艦テンプレートは*リミックス*するのに最適なものです；以下のエントリーは
*学ぶ*のに最適なものです。誰かのビデオやチュートリアルをここにキュレーションすることは、
プロジェクトを人々に届ける仕事に対する具体的で目に見える認定行為です ——
[表彰システムの支柱 ⑤](RECOGNITION-SYSTEM.md#pillar-5)の、
[`CONTRIBUTORS.md`](CONTRIBUTORS.md) と並ぶもう一方の半分。
すべてのエントリーはその著者をクレジットします。

## 基準（視聴数ではなく質をキュレートします）

以下の場合、一つの作品はここに載る資格があります：

- **現在の設計に正確** —— North Star を正しく教えます（フレームワークは LLM を
  実行しない；状態はファイルです；認証情報はローカルに留まる；federation は
  peer-to-peer です）。信頼モデルを誤って伝える洗練されたビデオは、ビデオが
  ない場合より害があり、掲載されません。
- **真に有用** —— 新参者がそれを見て、以前はできなかったことができるようになります：
  hub を立ち上げる、ワークフローを書く、federation リンクを繋げる。
- **最後まで見られる / 読める** —— 完成されており、ティーザーではありません。
- **明確に帰属表示されている** —— 著者をクレジットしリンクします。

ビュー数、フォロワー数、またはリファラル数では**ランク付けしません**。
federation を正しく説明する 200 ビューのビデオは、North Star を誤って伝える
50,000 ビューのビデオに勝ります。これはシステムの他の部分と同じです：
支柱 ① はテンプレートをランク付けし人々をランク付けしない；支柱 ⑤ は
*学ぶ価値があるもの*をキュレートし、視聴者数ではありません。

## 提出方法

**ビデオやチュートリアルを作りましたか？フロー全体は一つのフォーマット準拠の PR です** ——
余分な儀式も、他の場所に記入するフォームもありません。フォーマットでエントリーを追加し
PR を開いてください；それだけです。

1. **pull request を開いて**正しいセクションの下にエントリーを追加してください ——
   あなたのものでも他の人のものでも。[ビデオ](#ビデオ)の下の `<!-- コメント -->`
   のエントリーフォーマットを使ってください。
2. **maintainer が[基準](#基準視聴数ではなく質をキュレートします)に照らしてチェックします**
   —— 正確、有用、完成、帰属表示 —— そしてマージします（lazy consensus、
   [`GOVERNANCE.md`](../../GOVERNANCE.md#how-a-change-lands) 参照）。
3. **クレジットを得ます。** ここに掲載された作品は [`CONTRIBUTORS.md`](CONTRIBUTORS.md)
   の 📹 / 📢 / ✅ / 📝 の行も獲得します。ここにキュレーションされることとそこに
   記録されることは二つの異なる認定行為です。

## 誠実な境界

- **リンクしクレジットします；ホストも支払いもしません。** 掲載は*この特定の作品*が
  AipeHub をうまく教えるという編集上の認定です —— その著者が他の場所で言うまたは
  売るすべてのものへの支持ではありません。
- **エントリーは削除される可能性があります。** 設計が進むにつれて、教材は古くなります。
  現在の設計を正確に教えなくなったエントリーは、注記とともに削除します ——
  間違ったチュートリアルを掲載し続けることは、このページがまさに助けようとしている
  新参者を誤解させることになります。
- **このページに虚栄指標はありません。** ビュー数なし、「トレンド」なし、
  リーチによるランキングなし。陳列館は*学ぶ価値があるもの*を記録します、それだけです。
