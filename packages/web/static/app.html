    欢迎使用 AipeHub · 先看几条

    1. 这是个人本地工具,不是云服务。你的对话、画像、报告、API key 都存在你这台机器的 .aipehub-* 目录里。我们不收集任何数据,host 也不上报任何遥测。

    2. LLM 推理走第三方 API。你的 4 段自述会发给你配置的模型供应商(DeepSeek / Anthropic / OpenAI 等)做推理。供应商各自有自己的数据政策,看他们的隐私条款。如果有顾虑,可以用 mock provider 或本地 LLM。

    3. 这不是医生 / 心理咨询师 / 财务顾问 / 关系治疗师的替代品。个人成长教练们都被设计成"有边界的陪伴者"——触及红旗信号会让你去找专业人。不要把它们当成诊断或处方。

    4. 如果你正在心理危机中,请立即联系:

    • 🇨🇳 北京心理危机研究与干预中心:010-82951332
    • 🇨🇳 全国心理援助热线:400-161-9995
    • 🇲🇾 Befrienders Kuala Lumpur:03-7956 8144
    • 🌏 其他地区:findahelpline.com

    5. 你可以随时删除自己的数据。停掉 host,删除 .aipehub-* 目录即可彻底清掉。
