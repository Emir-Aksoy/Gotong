    工作区 API Key

    这里设置的 key 会被加密存到 <space>/secrets.enc.json；明文不写回任何 GET 响应。每个 agent 也可在自己的「编辑」表单里配私有 key，会优先生效。

    创建智能体

    ⚠️ 修改已存在的智能体可能影响进行中的会话；建议先停止再修改。

    工作流

    工作流把多个 agent 按顺序串起来。admin 派一个任务到 workflow 的触发能力，整个流程自动跑完。模板： templates/workflows/

    导入工作流

    aipehub.workflow/v1 格式。导入后立刻注册到 Hub，并写入 .aipehub/workflows/definitions/ 目录。

    导入

    支持 YAML 或 JSON。可上传 .yaml / .json 文件，或直接粘贴。
